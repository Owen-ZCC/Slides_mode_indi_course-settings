# 坪山区英语口语学习平台 - 英语学科功能需求

<product-goal>
  为小学低年段学生构建沉浸式、游戏化的英语听说环境，解决学生开口困难、语言环境缺失、互动频率低等问题。
</product-goal>

---

## 核心功能

<feature id="1" name="AI英语对话">
  <description>
    学生与AI角色进行实时英语语音对话，模拟真实语言交流场景
  </description>

  <capabilities>
    <capability name="场景化对话">
      由场景内配置的对话角色与学生进行语音对话
    </capability>
    <capability name="双模态输出">
      对话内容同时显示语音和英文文本
    </capability>
    <capability name="自适应难度">
      根据学生表现动态调整AI输出的词汇难度和句子复杂度
    </capability>
    <capability name="即时反馈">
      对话过程中对发音或内容进行即时鼓励性反馈
    </capability>
  </capabilities>
</feature>

<feature id="2" name="发音评估">
  <description>
    对学生英语发音进行多维度分析和记录
  </description>

  <capabilities>
    <capability name="流畅度检测">
      记录学生口语流畅度、卡顿频率
    </capability>
    <capability name="发音准确度">
      评估学生发音情况和错误类型
    </capability>
    <capability name="语调识别">
      识别录入语音的升降调、重音位置
    </capability>
  </capabilities>
</feature>

<feature id="3" name="对话记录与回放">
  <description>
    完整记录学生口语练习过程，支持回顾学习
  </description>

  <capabilities>
    <capability name="内容记录">
      记录口语对话内容和对话轮次
    </capability>
    <capability name="回放功能">
      支持回放单个学生的对话记录
    </capability>
    <capability name="薄弱点标记">
      系统自动标记需重点辅导的学生
    </capability>
  </capabilities>
</feature>

<feature id="4" name="学情数据看板">
  <description>
    实时汇总展示学生英语口语练习数据
  </description>

  <metrics>
    <metric>对话轮次</metric>
    <metric>卡顿频率</metric>
    <metric>错误类型分布</metric>
    <metric>流畅度趋势</metric>
  </metrics>
</feature>

<feature id="5" name="课程内容AI问答">
  <description>
    针对课程内容进行AI辅助问答，深化学习理解
  </description>

  <capabilities>
    <capability name="内容问答">
      对课程内容、工具回答结果进行AI对话
    </capability>
    <capability name="提问引导">
      提供提问建议选项，降低学生提问门槛
    </capability>
  </capabilities>
</feature>

## 教学场景示例

<scenarios>
  <scenario name="课堂口语练习">
    教师发布口语任务 → 学生与AI角色对话 → 教师实时查看数据 → 针对性辅导
  </scenario>

  <scenario name="课后自主练习">
    学生输入课程码 → 进入对话场景 → AI自适应调整难度 → 即时反馈鼓励
  </scenario>

  <scenario name="发音纠正">
    学生发音 → 系统识别语调重音 → 标记错误类型 → 提供改进建议
  </scenario>
</scenarios>

## AI能力要求

<ai-requirements>
  <requirement type="ASR">
    语音识别：支持儿童英语语音、升降调、重音位置识别
  </requirement>
  <requirement type="TTS">
    语音合成：自然流畅的英语角色语音输出
  </requirement>
  <requirement type="NLU">
    语义理解：理解学生表达意图，进行上下文连贯对话
  </requirement>
  <requirement type="Adaptive">
    自适应引擎：根据学生水平动态调整对话难度
  </requirement>
  <requirement type="Assessment">
    发音评估：流畅度、准确度、语调多维度评分
  </requirement>
</ai-requirements>
